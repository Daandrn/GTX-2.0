finfo_set_flags

finfo::set_flags

finfo_set_flags -- finfo::set_flags — Define as opções de configuração da libmagic

Descrição

Estilo procedural

finfo_set_flags(finfo $finfo, int $flags): bool

Estilo orientado a objetos

public finfo::set_flags(int $flags): bool

Esta função define várias opções de Fileinfo. As opções também podem ser definidas diretamente em finfo_open() ou em outras funções da Fileinfo.

Parâmetros

finfo

Uma instância de finfo, retornada por finfo_open().

flags

Uma ou disjunção de mais constantes Fileinfo.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.
