end — Set the internal pointer of an array to its last element

Descrição

end(array|object &$array): mixed

end() advances array's internal pointer to the last element, and returns its value.

Parâmetros

array

The array. This array is passed by reference because it is modified by the function. This means you must pass it a real variable and not a function returning an array because only actual variables may be passed by reference.

Valor Retornado

Returns the value of the last element or false for empty array.

Changelog

Versão Descrição
8.1.0 Chamar esta função em objects tornou-se obsoleto. Converta o object para um array usando get_mangled_object_vars() primeiro ou, em vez disso, use os métodos fornecidos por uma classe que implementa Iterator, como ArrayIterator.
7.4.0 Instâncias de classes SPL agora são tratadas como objetos vazios que não possuem propriedades em vez de chamar o método da interface Iterator com o mesmo nome desta função.

Exemplos

Exemplo #1 end() example

<?php

$fruits = array('apple', 'banana', 'cranberry');
echo end($fruits); // cranberry

?>
