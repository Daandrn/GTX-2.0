A classe CURLFile

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

Introdução

Esta classe ou CURLStringFile deve ser utilizada para fazer o upload de arquivos com CURLOPT_POSTFIELDS.

A desserialização de instancias CURLFile não é permitida. A partir do PHP 7.4.0, é explicitamente proíbida.

Sinopse da classe

class CURLFile {
/* Propriedades */
public string $name = "";
public string $mime = "";
public string $postname = "";
/* Métodos */
public setPostFilename(string $posted_filename): void
}

Propriedades

name

Nome do arquivo a ser feito o upload.

mime

MIME type do arquivo (o padrão é application/octet-stream).

postname

O nome do arquivo no campo de upload (o valor padrão é a propriedade name).
