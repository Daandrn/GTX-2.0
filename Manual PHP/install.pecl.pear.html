Compilando extensões PECL compartilhadas com o comando pecl

O PECL facilita a criação de extensões PHP compartilhadas. Usando o » comando pecl, faça o seguinte:


$ pecl install extname

Isso fará o download do código-fonte de extname, compilará e instalará extname.so em extension_dir. extname.so pode então ser carregada usando o php.ini.

Por padrão, o comando pecl não instalará pacotes marcados como alpha ou beta. Se nenhum pacote stable estiver disponível, um pacote beta pode ser instalado usando o seguinte comando:


$ pecl install extname-beta

Uma versão específica também pode ser instalada usando esta forma:


$ pecl install extname-0.1

Nota:

Depois de habilitar a extensão no php.ini, será necessário reiniciar o serviço web para que as alterações sejam detectadas.
