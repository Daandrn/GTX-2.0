A classe ReturnTypeWillChange

(PHP 8 >= 8.1.0)

Introdução

A maioria dos métodos internos non-final agora precisam de métodos de overriding para declarar um tipo de retorno compatível, caso contrário, um aviso obsoleto é emitido durante a validação da herança. Caso o tipo de retorno não possa ser declarado para um método de overriding devido a problemas de compatibilidade entre versões do PHP, um atributo #[\ReturnTypeWillChange] pode ser adicionado para silenciar o aviso de obsolescência.

Sinopse da classe

final class ReturnTypeWillChange {
/* Métodos */
public __construct()
}
