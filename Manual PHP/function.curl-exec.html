curl_exec

curl_exec — Executa uma sessão cURL

Descrição

curl_exec(CurlHandle $handle): string|bool

Execute a sessão cURL fornecida.

Esta função deve ser chamada após inicializar uma sessão cURL e todas as opções para a sessão são definidas.

Parâmetros

handle

Um manipulador cURL retornado por curl_init().

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha. No entanto, se a opção CURLOPT_RETURNTRANSFER estiver definida, ela retornará o resultado em caso de sucesso, false em caso de falha.

Aviso

Esta função pode retornar o valor booleano false, mas também pode retornar um valor não booleano que pode ser avaliado como false. Leia a seção sobre Booleanos para mais informações. Use o operador === para testar o valor retornado por esta função.

Nota:

Observe que os códigos de retorno que indicam erros (como por exemplo 404 Not found) não são tratados como falhas. Use curl_getinfo() para verificar esses casos.

Changelog

Versão Descrição
8.0.0 O parâmetro handle agora espera uma instância de CurlHandle; anteriormente, um resource era esperado.

Exemplos

Exemplo #1 Obtendo uma página da web

<?php
// Cria um novo recurso cURL
$ch = curl_init();

// Confiura a URL e opções
curl_setopt($ch, CURLOPT_URL, "http://www.example.com/");
curl_setopt($ch, CURLOPT_HEADER, 0);

// Obtem os dados
curl_exec($ch);

// Fecha o recurso e libera recursos internos
curl_close($ch);
?>
