is_infinite

is_infinite — Verifica se um número é infinito

Descrição

is_infinite(float $num): bool

Retorna informando se num é INF ou -INF.

Parâmetros

num

O float a verificar

Valor Retornado

true se num é INF ou -INF, senão retorna false.

Exemplos

Exemplo #1 is_infinite() example

<?php
$inf = 1e308 * 2;

var_dump($inf, is_infinite($inf));

$negative_inf = -$inf;

var_dump($negative_inf, is_infinite($negative_inf));
?>

O exemplo acima produzirá:

float(INF)
bool(true)
float(-INF)
bool(true)
