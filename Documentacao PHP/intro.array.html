Introdução

Estas funções permitem interagir e manipular arrays de várias formas. Arrays são essenciais para armazenar, gerenciar e operar sobre conjuntos de variáveis.

Arrays simples e multidimensionais são suportados e podem ser criados pelo usuário ou por outras funções. Existem funções específicas de manipulação de banco de dados para preencher arrays a partir de consultas a banco de dados, e diversas funções retornam arrays.

Consulte a seção Arrays do manual para uma explicação detalhada sobre como arrays são implementados e usados no PHP. Veja também Operadores de array para outras maneiras de manipular os arrays.
