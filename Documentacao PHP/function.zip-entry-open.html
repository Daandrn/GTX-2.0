zip_entry_open

zip_entry_open — Abre um arquivo para leitura

Aviso

Esta função tornou-se OBSOLETA a partir do PHP 8.0.0. O uso desta função é fortemente desencorajado.

Descrição

zip_entry_open(resource $zip_dp, resource $zip_entry, string $mode = "rb"): bool

Abre um arquivo para leitura.

Parâmetros

zip_dp

Um recurso válido retornado por zip_open().

zip_entry

Uma entrada de diretório retornada por zip_read().

mode

Algum dos modos especificados na documentação da fopen().

Nota:

Atualmente, o parâmetro mode é ignorado e o modo padrão suportado é "rb". Isto é devido ao fato que o suporte a zip no PHP é apenas de leitura.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Nota:

Diferentemente de fopen() e outras funções similares, o valor retornado de zip_entry_open() somente indica o resultado da operação, e não o valor em si não é necessário para a leitura ou fechamento recurso.

Changelog

Versão Descrição
8.0.0 Esta função está desencorajada em relação a API orientada a objeto.
