log

log — Logaritmo natural

Descrição

log(float $num, float $base = M_E): float

Se o parâmetro opcional base for informado, log() retorna logbase num, senão log() retorna o logaritmo natural de num.

Parâmetros

num

O valor para calcular o logaritmo

base

O logaritmo base opcional para usar (padrão é 'e' e assim o logaritmo natural).

Valor Retornado

O logarimo de num na base, se fornecido, ou o logaritmo natural.
