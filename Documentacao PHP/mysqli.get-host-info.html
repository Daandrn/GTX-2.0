mysqli::$host_info

mysqli_get_host_info

mysqli::$host_info -- mysqli_get_host_info — Returns a string representing the type of connection used

Descrição

Estilo orientado a objetos

string $mysqli->host_info;

Estilo procedural

mysqli_get_host_info(mysqli $mysql): string

Returns a string describing the connection represented by the mysql parameter (including the server host name).

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

Valor Retornado

A character string representing the server hostname and the connection type.

Exemplos

Exemplo #1 $mysqli->host_info example

Estilo orientado a objetos

<?php

mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);
$mysqli = new mysqli("localhost", "my_user", "my_password", "world");

/* print host information */
printf("Host info: %s\n", $mysqli->host_info);

Estilo procedural

<?php

mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);
$link = mysqli_connect("localhost", "my_user", "my_password", "world");

/* print host information */
printf("Host info: %s\n", mysqli_get_host_info($link));

Os exemplos acima produzirão:

Host info: Localhost via UNIX socket
