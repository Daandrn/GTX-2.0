zip_read

zip_read — Lê o próximo item em um arquivo ZIP

Aviso

Esta função tornou-se OBSOLETA a partir do PHP 8.0.0. O uso desta função é fortemente desencorajado.

Descrição

zip_read(resource $zip): resource|false

Lê o próximo item dentro de uma arquivo ZIP.

Parâmetros

zip

Um arquivo ZIP anteriormente aberto com zip_open().

Valor Retornado

Retorna uma conexao com o arquivo ZIP para ser usado com as funções zip_entry_..., ou false se não houver arquivos para serem lidos, ou um código de erro caso ocorra algum.

Changelog

Versão Descrição
8.0.0 Essa função está desencorajada em relação à API orientada a objetos, veja ZipArchive::statIndex().
