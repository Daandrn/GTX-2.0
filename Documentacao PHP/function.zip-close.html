zip_close

zip_close — Fecha um arquivo ZIP

Aviso

Esta função tornou-se OBSOLETA a partir do PHP 8.0.0. O uso desta função é fortemente desencorajado.

Descrição

zip_close(resource $zip): void

Fecha um dado arquivo ZIP.

Parâmetros

zip

Um arquivo ZIP previamente aberto com zip_open().

Valor Retornado

Nenhum valor é retornado.

Changelog

Versão Descrição
8.0.0 Esta função está obsoleta, em relação a API orientada a objetos, veja ZipArchive::close().
