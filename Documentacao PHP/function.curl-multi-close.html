curl_multi_close

curl_multi_close — Fechar um conjunto de alças cURL

Descrição

curl_multi_close(CurlMultiHandle $multi_handle): void

Nota:

Esta função não tem efeito. Antes do PHP 8.0.0, esta função era usada para fechar o recurso.

Fecha um conjunto de alças cURL.

Parâmetros

multi_handle

Um manipulador cURL múltiplo retornado por curl_multi_init().

Valor Retornado

Nenhum valor é retornado.

Changelog

Versão Descrição
8.0.0 O parâmetro multi_handle agora espera uma instância de CurlMultiHandle; anteriormente, um resource era esperado.

Exemplos

Exemplo #1 curl_multi_close() example

Este exemplo criará dois identificadores cURL, os adicionará a um identificador múltiplo e os executará em paralelo.

<?php
// Cria um recurso cURL
$ch1 = curl_init();
$ch2 = curl_init();

// Configura a URL e outras opções
curl_setopt($ch1, CURLOPT_URL, "http://www.example.com/");
curl_setopt($ch1, CURLOPT_HEADER, 0);
curl_setopt($ch2, CURLOPT_URL, "http://www.php.net/");
curl_setopt($ch2, CURLOPT_HEADER, 0);

// Cria um recurso cURL múltiplo
$mh = curl_multi_init();

// Adiciona dois manipuladores
curl_multi_add_handle($mh,$ch1);
curl_multi_add_handle($mh,$ch2);

// Executa o manipulador múltipo
do {
$status = curl_multi_exec($mh, $active);
if ($active) {
curl_multi_select($mh);
}
} while ($active && $status == CURLM_OK);

//close the handles
curl_multi_remove_handle($mh, $ch1);
curl_close($ch1);
curl_multi_remove_handle($mh, $ch2);
curl_close($ch2);
curl_multi_close($mh);

?>
