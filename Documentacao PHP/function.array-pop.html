array_pop

array_pop — Extrai um elemento do final do array

Descrição

array_pop(array &$array): mixed

array_pop() extrai e retorna o valor do último elemento de array, diminuindo o array em um elemento.

Nota: Após o uso desta função, assim como na função reset(), o ponteiro interno do array fornecido apontará para o primeiro elemento.

Parâmetros

array

O array para obter os valores.

Valor Retornado

Retorna o valor do último elemento de array. Se array é vazio, null será retornado.

Exemplos

Exemplo #1 Exemplo da array_pop()

<?php
$stack = array("orange", "banana", "apple", "raspberry");
$fruit = array_pop($stack);
print_r($stack);
?>

Depois disso, $stack irá conter apenas 3 elementos: 

Array
(
    [0] => orange
    [1] => banana
    [2] => apple
)

e raspberry será passado para $fruit.
