method_exists

method_exists — Checks if the class method exists

Descrição

method_exists(object|string $object_or_class, string $method): bool

Checks if the class method exists in the given object_or_class.

Parâmetros

object_or_class

An object instance or a class name

method

The method name

Valor Retornado

Returns true if the method given by method has been defined for the given object_or_class, false otherwise.

Exemplos

Exemplo #1 method_exists() example

<?php
$directory = new Directory('.');
var_dump(method_exists($directory,'read'));
?>

O exemplo acima produzirá:

bool(true)

Exemplo #2 Static method_exists() example

<?php
var_dump(method_exists('Directory','read'));
?>

O exemplo acima produzirá:

bool(true)

Notas

Nota:

Usar esta função fará com que qualquer autoloader registrado seja usado se a classe ainda não for conhecida.
